# Overview

EduCloud - The Cloud-Based Student Management System is a fully functional SaaS web application developed for managing students, courses, authentication, and cloud-based file storage. 
<br>It provides an end-to-end platform where students can register, log in, manage their enrolled courses, and track their academic progress — all hosted on the cloud.
<br>The project demonstrates real-world cloud computing concepts like cloud deployment, user authentication, database integration, and role-based access control using AWS services along with full-stack web development technologies.

# Features
- Student Registration and Login
- Student Dashboard
- Admin Panel
- Course Management
- Role-Based Access Control
- File Upload System
- Cloud Deployment using AWS
- Real-Time Data Management
- Secure Authentication System
